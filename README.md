This program will track each game you play on the EA NHL games using screen shots.
The Programs purpose is to allow players to play EA NHL games with a more analytical approach.
Calculations will be made on each players deffensive and offensive impact and be portrayed on a point graph.
Stats will be taken from uploaded screenshots.
AI will read the screenshots and seperate the stats into their proper place
This allows the application to be used in any Hockey game.

Disclaimer: People must use their own API key!
